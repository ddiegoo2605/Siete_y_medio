Tutorial

¿Como se juega el programa?

Opcion 0.1: Add/Remove/Show Players

Opcion 1.1: New Human Player

Lo primero de todo te pedira poner el nombre, pero no puede estar vacio o que tenga letras o numeros.

Después te pedira de poner un DNI, pero tiene que ser uno valido.

Y para acabar tienes que poner obligatorio que tipo de perfil de riesgo quieres que son:

Atrevido | Bold

Normal | Moderated

Prudente | Cautious

Entonces después de todo esto te mostrara toda la informacion que has puesto y te dira si esta bien para guardarlo o si no lo quieres guardar.

Opcion 1.2: New Boot

Para empezar sera como en la opcion 1.1 pero después de ponerlo te generara el dni y para acabar sera lo mismo que en la opcion 1.1, decir que perfil quieres y si esta lo quieres guardar.

Opcion 1.3: Show/Remove Players

Se mostraran en una tabla todos los jugadores y bots disponibles. Entonces en esa opcion podras eliminarlos.
